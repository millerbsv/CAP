Proyecto del centro de automatización de procesos de la pontificia universidad javeriana Cali, en donde se pretende por medio del arduino yun programar robots seriales por medio de Ethernet para reactivar las labores de los mismo.

Se crea una aplicación web que reconoce la sintaxis de la programación de los robots, luego la aplicación se comunicara con el arduino yun mediante servicios web, luego el arduino yun enviara la información del programa al robot para que finamente el robot trabaje.
